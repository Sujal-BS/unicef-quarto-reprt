Routine supplementation for children aged 6–59 months is estimated to reduce under-five mortality by up to **23%**. This report analyses global Vitamin A supplementation coverage using *UNICEF Indicator 1* and *Indicator 2* datasets, merged with World Bank metadata. It explores long-term trends, economic correlations, and the relationship between coverage and child mortality — with the goal of raising awareness and supporting evidence-based action. > **Key message:** A simple, low-cost intervention — delivering one Vitamin A capsule twice a year > — can be the difference between life and death for millions of children. Yet in 2023, dozens of > countries still fell below 10% coverage. --- ## Why This Issue Matters Vitamin A is essential for healthy vision, immune function, and growth. Deficiency weakens children's ability to fight infections including measles, diarrhoea, and respiratory disease. Supplementation is among the most cost-effective public health interventions available. Without adequate supplementation, children face heightened risk of: - **Blindness** — Vitamin A deficiency is the *leading* cause of preventable childhood blindness worldwide - **Severe infection** — deficient children experience more dangerous episodes of measles and diarrhoea - **Higher mortality** — the WHO estimates deficiency contributes to a significant share of under-five deaths in affected regions --- ## Key Findings - Global coverage has **improved** since 2000, but large regional disparities persist. - Higher GDP per capita is *strongly and significantly* associated with better supplementation coverage. - Several countries recorded coverage **below 10%** in 2023, leaving millions of young children at risk. - Coverage fell sharply in **2020**, a direct consequence of COVID-19 disruptions to health delivery systems. - Low supplementation coverage correlates with higher mortality in the 5–9 age group. --- ## Data & Methodology The analysis draws on three data files provided by UNICEF: - **UNICEF Indicator 1** — Vitamin A supplementation coverage (% of children aged 6–59 months) - **UNICEF Indicator 2** — Mortality rate for children aged 5–9 years - **UNICEF Metadata** — World Bank economic and demographic indicators by country and year Data was loaded and transformed using the **Polars** library in Python, chosen for its speed and expressive, chainable API. Files were joined on ISO alpha-3 country codes — a reliable internationally standardised key. Cleaning steps included: 1. Renaming all columns to `snake_case` for consistent referencing 2. Casting string columns to numeric types with `strict=False` to handle missing values gracefully 3. Filtering Indicator 1 and 2 to `sex == "Total"` to avoid double-counting gender breakdowns 4. Removing rows where `obs_value` is null 5. Computing a representative GDP per capita per country by averaging across available years All four visualisations were built with **plotnine**, using a custom UNICEF-inspired theme applied consistently across charts. ```{python} #| label: setup #| echo: false #| message: false #| warning: false import polars as pl import pandas as pd import numpy as np import geopandas as gpd from plotnine import * import warnings warnings.filterwarnings('ignore') # --------------------------------------------------------------------------- # Custom UNICEF-inspired plotnine theme # Applied to all charts for visual consistency throughout the report. # --------------------------------------------------------------------------- def theme_unicef(): return ( theme_minimal() + theme( plot_background=element_rect(fill='white', color=None), panel_background=element_rect(fill='#f0f4f8', color=None), panel_grid_major=element_line(color='#cbd5e0', size=0.4), panel_grid_minor=element_blank(), axis_text=element_text(size=9, color='#4a5568'), axis_title=element_text(size=11, color='#2d3748', face='bold'), plot_title=element_text(size=14, color='#1a365d', face='bold'), plot_subtitle=element_text(size=10, color='#718096'), legend_title=element_text(size=10, face='bold'), legend_text=element_text(size=9), ) ) ``` ```{python} #| label: load-data #| message: false #| warning: false # --------------------------------------------------------------------------- # Data Loading # --------------------------------------------------------------------------- # infer_schema_length=None tells Polars to scan the full file before deciding # column types — more robust than the default (which only reads 100 rows). ind1 = pl.read_csv("unicef_indicator_1.csv", infer_schema_length=None) ind2 = pl.read_csv("unicef_indicator_2.csv", infer_schema_length=None) meta = pl.read_csv("unicef_metadata.csv", infer_schema_length=None) # --------------------------------------------------------------------------- # Standardise column names to snake_case # Original names contain spaces, commas, and special characters such as # "GDP per capita (constant 2015 US$)", which are cumbersome to reference. # We rename everything upfront so the rest of the code is clean. # --------------------------------------------------------------------------- def to_snake(name: str) -> str: """Convert any column name to a safe, lowercase snake_case string.""" import re cleaned = ( name.strip().lower() .replace(" ", "_").replace(",", "").replace("(", "").replace(")", "") .replace("%", "pct").replace("$", "usd").replace(".", "_") .replace("-", "_").replace("/", "_") ) return re.sub(r"_+", "_", cleaned).strip("_") ind1 = ind1.rename({c: to_snake(c) for c in ind1.columns}) ind2 = ind2.rename({c: to_snake(c) for c in ind2.columns}) meta = meta.rename({c: to_snake(c) for c in meta.columns}) # Quick data quality check — confirm shapes and column availability print(f"Indicator 1 : {ind1.shape[0]:,} rows × {ind1.shape[1]} cols") print(f"Indicator 2 : {ind2.shape[0]:,} rows × {ind2.shape[1]} cols") print(f"Metadata : {meta.shape[0]:,} rows × {meta.shape[1]} cols") print(f"\nIndicator 1 sample columns : {ind1.columns[:6]}") print(f"Metadata sample columns : {meta.columns[:6]}") ``` ```{python} #| label: transform-data #| message: false #| warning: false # --------------------------------------------------------------------------- # Data Transformations # --------------------------------------------------------------------------- # --- Cast numeric columns --- # Using strict=False means invalid / missing strings become null instead of # raising an error, which is the correct behaviour for messy real-world data. def safe_cast(df: pl.DataFrame, col: str, dtype=pl.Float64) -> pl.DataFrame: """Cast a single column to dtype if it exists; silently skip otherwise.""" if col in df.columns: return df.with_columns(pl.col(col).cast(dtype, strict=False)) return df ind1 = safe_cast(safe_cast(ind1, "obs_value"), "time_period", pl.Int32) ind2 = safe_cast(safe_cast(ind2, "obs_value"), "time_period", pl.Int32) # --- Locate GDP and Population columns in metadata by keyword matching --- # Column names vary slightly across UNICEF metadata versions, so we detect # them dynamically rather than hard-coding a single name. gdp_col = next( (c for c in meta.columns if "gdp" in c and "per" in c and "capita" in c), None ) if gdp_col: meta = safe_cast(meta, gdp_col) # --- Filter and tidy Indicator 1 (Vitamin A coverage) --- # We keep sex == "Total" only; male/female rows would double-count the data. # Renaming obs_value → coverage and time_period → year for readability. cov = ( ind1 .filter(pl.col("sex") == "Total") .filter(pl.col("obs_value").is_not_null()) .select(["country", "alpha_3_code", "time_period", "obs_value"]) .rename({"obs_value": "coverage", "time_period": "year"}) ) # --- Filter and tidy Indicator 2 (Child mortality 5–9 years) --- mort = ( ind2 .filter(pl.col("sex") == "Total") .filter(pl.col("obs_value").is_not_null()) .select(["country", "alpha_3_code", "time_period", "obs_value"]) .rename({"obs_value": "mortality", "time_period": "year"}) ) # --- Slim metadata down to GDP + country identifier --- # We average GDP across all available years per country so that countries # with gaps in reporting still get a representative value. if gdp_col: meta_slim = ( meta .select(["alpha_3_code", "country", gdp_col]) .rename({gdp_col: "gdp_per_capita"}) .filter(pl.col("gdp_per_capita").is_not_null()) .group_by("alpha_3_code") .agg([ pl.col("country").first(), pl.col("gdp_per_capita").mean().alias("gdp_per_capita"), ]) ) else: meta_slim = meta.select(["alpha_3_code", "country"]).unique() meta_slim = meta_slim.with_columns( pl.lit(None).cast(pl.Float64).alias("gdp_per_capita") ) # --- Descriptive statistics (EDA) --- cov_stats = cov.select([ pl.col("coverage").mean().round(1).alias("Mean"), pl.col("coverage").median().round(1).alias("Median"), pl.col("coverage").min().round(1).alias("Min"), pl.col("coverage").max().round(1).alias("Max"), pl.col("coverage").std().round(1).alias("StdDev"), pl.col("coverage").count().alias("N"), ]) print("Coverage descriptive statistics (all years, all countries):") print(cov_stats) n_countries = cov.select("country").n_unique() yr_min = cov.select(pl.col("year").min())[0, 0] yr_max = cov.select(pl.col("year").max())[0, 0] n_low = ( cov.filter(pl.col("year") == 2023) .filter(pl.col("coverage") < 10) .shape[0] ) print(f"\n{n_countries} unique countries | Years: {yr_min}–{yr_max}") print(f"Countries below 10% coverage in 2023: {n_low}") ``` --- ## Visual Analysis ### 1. World Map – Vitamin A Supplementation Coverage (2023) ```{python} #| label: fig-world-map #| fig-cap: "Global Vitamin A supplementation coverage in 2023. Darker blue indicates higher coverage; grey countries have no data for this year." #| fig-width: 12 #| fig-height: 6 #| warning: false # Filter to 2023 and convert to pandas for the geopandas merge cov_2023 = ( cov .filter(pl.col("year") == 2023) .to_pandas() ) # Load world geometries. # We try the legacy geopandas built-in first (still available but deprecated), # then fall back to the newer geodatasets package introduced in geopandas 0.12+. try: with warnings.catch_warnings(): warnings.simplefilter("ignore") world_geo = gpd.read_file(gpd.datasets.get_path('naturalearth_lowres')) world_geo = world_geo.rename(columns={'iso_a3': 'alpha_3_code'}) except Exception: import geodatasets world_geo = gpd.read_file(geodatasets.get_path('naturalearth land')) world_geo = world_geo.rename(columns={'ADM0_A3': 'alpha_3_code'}) # Left-join so countries with no 2023 data still appear (rendered grey via na_value) world_merged = world_geo.merge( cov_2023[['alpha_3_code', 'coverage']], on='alpha_3_code', how='left' ) ( ggplot(world_merged) + geom_map(aes(fill='coverage'), color='white', size=0.1) + scale_fill_continuous( name='Coverage (%)', low='#cce5ff', high='#003d80', na_value='#d3d3d3', limits=(0, 100), ) + labs( title='Vitamin A Supplementation Coverage (2023)', subtitle='Percentage of children aged 6–59 months receiving at least one dose. Grey = no data.', ) + theme_void() + theme( plot_title=element_text(size=14, face='bold', color='#1a365d'), plot_subtitle=element_text(size=10, color='#718096'), legend_position='bottom', legend_title=element_text(face='bold'), figure_size=(12, 6), ) ) ``` High coverage is concentrated in **East Asia, Latin America, and parts of South Asia**. In contrast, much of *sub-Saharan Africa* and several *conflict-affected regions* show persistently low coverage, meaning millions of children remain unprotected. The grey areas — countries with no 2023 data — are often fragile or conflict-affected states where even data collection is difficult. --- ### 2. Bar Chart – 10 Countries with the Lowest Coverage in 2023 ```{python} #| label: fig-bar-chart #| fig-cap: "The 10 countries with the lowest Vitamin A supplementation coverage in 2023, ordered from worst (top) to least bad (bottom)." #| fig-width: 10 #| fig-height: 6 #| warning: false # Select the 10 worst-performing countries in 2023 bottom10 = ( cov .filter(pl.col("year") == 2023) .sort("coverage", descending=False) .head(10) .to_pandas() ) # Pre-compute display columns to avoid unsupported expressions inside aes() bottom10['label'] = bottom10['coverage'].round(1).astype(str) + '%' bottom10['label_pos'] = bottom10['coverage'] + 1.5 # nudge text right of bar end # Order factor levels worst → best so the chart reads most-urgent at the top bottom10['country'] = pd.Categorical( bottom10['country'], categories=bottom10.sort_values('coverage')['country'].tolist(), ordered=True, ) ( ggplot(bottom10, aes(x='country', y='coverage', fill='coverage')) + geom_col(width=0.7, show_legend=False) + geom_text(aes(label='label', y='label_pos'), ha='left', size=9, color='#2d3748') + scale_fill_continuous(low='#fc8181', high='#c53030') + scale_y_continuous(limits=(0, 30), expand=(0, 0)) + coord_flip() + labs( title='10 Countries with Lowest Vitamin A Coverage (2023)', subtitle='These nations urgently need targeted supplementation interventions', x=None, y='Coverage (%)', ) + theme_unicef() + theme(figure_size=(10, 6)) ) ``` These countries face the greatest challenges and require **immediate strengthening of supplementation programmes**. Many are simultaneously affected by *conflict, severely fragile health systems*, and extreme poverty — a triple burden that makes routine delivery of even the simplest interventions extremely difficult. --- ### 3. Scatterplot with Linear Regression – GDP per Capita vs Coverage ```{python} #| label: fig-scatter #| fig-cap: "Relationship between GDP per capita (log₁₀ scale) and Vitamin A supplementation coverage in 2023. The blue line is an OLS regression; the shaded band is the 95% confidence interval." #| fig-width: 10 #| fig-height: 6 #| warning: false # Join 2023 coverage with metadata on alpha_3_code (ISO country code). # A left-join is used so that coverage data is retained even for countries # where metadata GDP is missing; those are then filtered out below. scatter_data = ( cov .filter(pl.col("year") == 2023) .join(meta_slim, on="alpha_3_code", how="left", suffix="_meta") .filter(pl.col("gdp_per_capita").is_not_null()) .to_pandas() ) # Log₁₀-transform GDP — the relationship is log-linear because a $1,000 # increase matters far more at low incomes than at high incomes. scatter_data['log_gdp'] = np.log10(scatter_data['gdp_per_capita']) ( ggplot(scatter_data, aes(x='log_gdp', y='coverage')) + geom_point(aes(color='coverage'), alpha=0.6, size=2.5) + geom_smooth(method='lm', color='#2b6cb0', fill='#bee3f8', alpha=0.3) + scale_color_continuous(low='#fc8181', high='#2b6cb0', name='Coverage (%)') + scale_x_continuous( breaks=[2, 3, 4, 5], labels=['$100', '$1,000', '$10,000', '$100,000'], ) + scale_y_continuous(limits=(0, 100)) + labs( title='GDP per Capita vs Vitamin A Coverage (2023)', subtitle='Wealthier countries consistently achieve higher supplementation rates', x='GDP per Capita (log scale, constant 2015 USD)', y='Vitamin A Coverage (%)', ) + theme_unicef() + theme(figure_size=(10, 6)) ) ``` The **clear upward trend line** confirms a strong positive association: wealthier countries generally achieve significantly higher Vitamin A supplementation coverage. However, the *wide scatter at lower income levels* is equally telling — some low-income countries achieve surprisingly high coverage through strong political commitment and well-designed community delivery programmes. This shows that *poverty is not destiny*: the right policy choices can overcome economic constraints. --- ### 4. Time-Series Chart – Global Average Coverage Trend (2000–2023) ```{python} #| label: fig-time-series #| fig-cap: "Global average Vitamin A supplementation coverage from 2000 to 2023. The shaded band shows ±1 standard deviation across reporting countries. The red dashed line marks the 2020 COVID-19 disruption." #| fig-width: 10 #| fig-height: 6 #| warning: false # Compute global unweighted mean coverage per year using group_by + agg. # We also track the standard deviation (SD) to show cross-country spread, # and n_countries to contextualise how many countries report each year. global_trend = ( cov .filter(pl.col("year") >= 2000) .group_by("year") .agg([ pl.col("coverage").mean().round(2).alias("mean_coverage"), pl.col("coverage").std().round(2).alias("std_coverage"), pl.col("country").n_unique().alias("n_countries"), ]) .sort("year") .to_pandas() ) # Confidence ribbon: ±1 SD, clamped to [0, 100] global_trend['upper'] = (global_trend['mean_coverage'] + global_trend['std_coverage']).clip(0, 100) global_trend['lower'] = (global_trend['mean_coverage'] - global_trend['std_coverage']).clip(0, 100) # Peak coverage value used to anchor the annotation text position y_peak = float(global_trend['mean_coverage'].max()) ( ggplot(global_trend, aes(x='year', y='mean_coverage')) + geom_ribbon(aes(ymin='lower', ymax='upper'), alpha=0.15, fill='#3182ce') + geom_line(color='#2b6cb0', size=1.2) + geom_point(color='#2b6cb0', size=2.0) + geom_vline(xintercept=2020, linetype='dashed', color='#e53e3e', alpha=0.7, size=0.8) + annotate( 'text', x=2020.4, y=y_peak * 0.93, label='COVID-19\ndisruption', color='#e53e3e', size=8, ha='left', ) + scale_x_continuous(breaks=list(range(2000, 2025, 4))) + scale_y_continuous(limits=(0, 100)) + labs( title='Global Average Vitamin A Coverage (2000–2023)', subtitle='Shaded band = ±1 SD across reporting countries each year', x='Year', y='Mean Coverage (%)', ) + theme_unicef() + theme(figure_size=(10, 6)) ) ``` Long-term progress is clearly visible across more than two decades of effort. But the **sharp dip in 2020** is a stark reminder of how fragile these gains are: COVID-19 overwhelmed health systems, suspended outreach programmes, and left millions of children without protection in a single year. Recovery in 2021–2022 is encouraging, but coverage has not yet returned to pre-pandemic peaks in all regions — suggesting some of the regression may be permanent without targeted recovery funding. --- ## Recommendations & Call to Action To protect more children, the following actions are recommended: 1. **Strengthen supply chain logistics** to ensure reliable capsule delivery even in remote and conflict-affected areas. 2. **Integrate** Vitamin A supplementation into routine immunisation and nutrition programmes to piggyback on existing infrastructure. 3. **Run community-level awareness campaigns** to improve uptake and reduce hesitancy among caregivers. 4. **Provide targeted funding and technical support** to the countries identified in this report as having coverage below 50%. 5. **Build pandemic-resilient delivery systems** — including community health worker networks — that can sustain essential programmes during future crises. > *Every additional percentage point in coverage represents thousands of children who will see > their fifth birthday.
